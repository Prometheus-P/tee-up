# Screen: Admin Login (관리자 로그인)

**Route**: `/admin/login`
**Status**: Implemented
**Last Updated**: 2025-12-18
**Priority**: P0
**Dependencies**: Admin Auth

---

## 1. 화면 개요

### 목적
플랫폼 관리자가 관리자 대시보드에 접근하기 위해 로그인한다.

### 사용자 흐름
```
/admin (미인증) → /admin/login → /admin (관리자 대시보드)
```

### 접근 조건
- **인증 필요**: No
- **권한 수준**: guest (관리자 자격 확인)
- **리디렉션**: 이미 관리자로 로그인 시 `/admin`으로 리디렉션

## 3. 컴포넌트 목록

### 3.1 AdminLoginForm

**폼 필드**

| Field | Type | Validation |
|-------|------|------------|
| email | email | required, admin email domain |
| password | password | required |

**특이사항**
- 카카오 로그인 불가 (이메일/비밀번호만)
- 실패 시 상세 메시지 숨김 (보안)

---

## 4. 데이터 요구사항

### Server Actions

| Action | 설명 |
|--------|------|
| `adminSignIn` | 관리자 로그인 (role 확인 포함) |

### 권한 확인

```typescript
// 로그인 후 role 확인
if (user.role !== 'admin') {
  throw new Error('Unauthorized');
}
```
