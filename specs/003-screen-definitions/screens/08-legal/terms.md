**Route**: `/terms`
**Status**: Implemented
**Last Updated**: 2025-12-18
**Priority**: P1
**Dependencies**: None

---

## 1. 화면 개요

### 목적
플랫폼 이용약관을 제공하여 법적 요구사항을 충족한다.

### 사용자 흐름
```
회원가입 → 이용약관 링크 클릭 → /terms
Footer → /terms
```

### 접근 조건
- **인증 필요**: No
- **권한 수준**: guest

## 3. 주요 컴포넌트

### 3.1 LegalPageHeader

**법률 페이지 헤더**

| Element | Description |
|---------|-------------|
| 제목 | "이용약관" |
| 업데이트 날짜 | "마지막 업데이트: YYYY년 M월 D일" |

### 3.2 TableOfContents

**목차**

| Element | Description |
|---------|-------------|
| 목록 | 섹션 링크 (앵커) |
| 스타일 | `--tee-surface` 배경, 좌측 border |

### 3.3 LegalContent

**약관 본문**

| Element | Description |
|---------|-------------|
| 제목 (h2) | "제N조 (조항명)" |
| 본문 | 약관 내용 |
| 하위 항목 | 번호 매긴 리스트 |

## 4. 콘텐츠 구조

### 주요 섹션

| 섹션 | 내용 |
|------|------|
| 총칙 | 목적, 정의, 약관 효력 |
| 서비스 이용 | 서비스 범위, 변경, 중단 |
| 회원가입 및 계정 | 가입 조건, 계정 관리, 탈퇴 |
| 프로 회원 특별조항 | 승인 절차, 포트폴리오, 리드 |
| 결제 및 환불 | 결제 방법, 환불 정책, 수수료 |
| 금지행위 | 금지 사항 목록 |
| 책임 제한 | 면책 조항 |
| 분쟁 해결 | 준거법, 관할법원 |
