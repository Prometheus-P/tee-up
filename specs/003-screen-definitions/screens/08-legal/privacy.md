## 1. 화면 개요

### 목적
개인정보 수집 및 처리에 관한 방침을 제공하여 법적 요구사항을 충족한다.

### 사용자 흐름
```
회원가입 → 개인정보처리방침 링크 클릭 → /privacy
Footer → /privacy
```

### 접근 조건
- **인증 필요**: No
- **권한 수준**: guest

## 3. 주요 컴포넌트

### 3.1 LegalPageHeader

**법률 페이지 헤더**

| Element | Description |
|---------|-------------|
| 제목 | "개인정보처리방침" |
| 업데이트 날짜 | "마지막 업데이트: YYYY년 M월 D일" |

### 3.2 TableOfContents

**목차**

| Element | Description |
|---------|-------------|
| 목록 | 섹션 링크 (앵커) |
| 스타일 | `--tee-surface` 배경, 좌측 border |

### 3.3 DataTable

**수집 항목 테이블**

| Column | Description |
|--------|-------------|
| 항목 | 수집하는 개인정보 |
| 필수/선택 | 필수 또는 선택 여부 |
| 수집 목적 | 해당 정보의 이용 목적 |

## 4. 콘텐츠 구조

### 주요 섹션

| 섹션 | 내용 |
|------|------|
| 개인정보 수집 항목 | 필수/선택 항목 목록 |
| 수집 및 이용 목적 | 회원관리, 서비스 제공, 마케팅 |
| 보유 및 이용 기간 | 회원 탈퇴 시 즉시 파기, 법적 보관 기간 |
| 제3자 제공 | 제공 대상, 항목, 목적 |
| 개인정보 처리 위탁 | 위탁 업체, 위탁 업무 |
| 이용자의 권리 | 열람, 정정, 삭제, 처리정지 요구권 |
| 쿠키 사용 | 쿠키 목적, 관리 방법 |
| 개인정보 보호책임자 | 연락처 정보 |
