# Screen: Reset Password (비밀번호 재설정)

**Route**: `/auth/reset-password`
**Status**: Implemented
**Last Updated**: 2025-12-18
**Priority**: P1
**Dependencies**: Supabase Auth

---

## 1. 화면 개요

### 목적
사용자가 비밀번호를 잊어버렸을 때 이메일로 재설정 링크를 받는다.

### 사용자 흐름
```
/auth/login → /auth/reset-password → 이메일 발송 → 이메일 링크 클릭 → 새 비밀번호 설정
```

### 접근 조건
- **인증 필요**: No
- **권한 수준**: guest

---

## 2. 레이아웃 구조

### Step 1: 이메일 입력

```
┌──────────────────┐
│ 비밀번호 재설정   │
│                  │
│ 가입 시 사용한    │
│ 이메일을 입력해주세요│
│                  │
│ [이메일 입력    ] │
│                  │
│ [재설정 링크 발송]│
│                  │
│ [로그인으로 돌아가기]│
└──────────────────┘
```

### Step 2: 이메일 발송 완료

```
┌──────────────────┐
│ 이메일을 확인하세요│
│                  │
│ {email}로        │
│ 재설정 링크를     │
│ 보내드렸습니다.   │
│                  │
│ [로그인으로 돌아가기]│
└──────────────────┘
```
