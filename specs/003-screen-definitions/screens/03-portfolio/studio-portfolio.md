# Screen: Studio Portfolio (스튜디오 포트폴리오)

**Route**: `/studio/[studioSlug]`
**Status**: Implemented
**Last Updated**: 2025-12-18
**Priority**: P1
**Dependencies**: Portfolio System

---

## 1. 화면 개요

### 목적
골프 아카데미/스튜디오의 공개 페이지로, 소속 프로 목록과 스튜디오 정보를 보여준다.

### 사용자 흐름
```
검색/공유 → /studio/[studioSlug] → 프로 선택 → /[slug]
```

### 접근 조건
- **인증 필요**: No
- **권한 수준**: guest

---

## 2. 레이아웃 구조

│ 설명: "최고의 시설과 프로들이 있는 골프 아카데미"            │
├────────────────────────────────────────────────────────────┤
│ 소속 프로 목록                                               │
│ ┌─────────┐ ┌─────────┐ ┌─────────┐ ┌─────────┐            │

---

## 3. 주요 컴포넌트

- `StudioHero` - 스튜디오 히어로 섹션
- `StudioInfo` - 위치, 설명
- `ProCard` - 소속 프로 카드 (그리드)
- `SocialLinks` - 소셜 미디어 링크

---

## 4. 데이터 요구사항

### Server Actions

| Action | 설명 |
|--------|------|
| `getStudioBySlug` | 스튜디오 정보 조회 |
| `getStudioPros` | 소속 프로 목록 조회 |
