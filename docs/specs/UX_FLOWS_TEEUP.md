# TEE:UP 사용자 경험(UX) 흐름 정의 v1.0

본 문서는 TEE:UP 웹 서비스의 핵심 사용자 흐름 3가지에 대해 Apple Human Interface Guidelines (HIG) 기반의 UX 패턴과 화면 구성을 정의합니다. 앞서 정의된 디자인 원칙, 레이아웃 가이드, 컴포넌트 라이브러리를 참조하여 일관되고 직관적인 사용자 경험을 제공하는 것을 목표로 합니다.

## 1. Golfer Home (Directory) → Pro Detail Profile → Initiate Inquiry (골퍼: 프로 찾기 및 문의)

**플로우 요약**: 골퍼가 홈/디렉토리 페이지에서 원하는 프로를 찾아 상세 프로필을 확인한 후, 1:1 문의를 시작하는 과정입니다.

### 1.1. Screen 1.1: 골퍼 홈 (프로 디렉토리)

-   **핵심 목적**: 골퍼가 다양한 프로들의 목록을 탐색하고, 자신에게 맞는 프로를 검색/필터링하여 발견합니다.
-   **한 줄 요약**: "프리미엄 골프 프로들의 매거진형 디렉토리"
-   **사용 핵심 컴포넌트**: `Header`, `Card` (프로 스니펫), `Input` (검색), `Tag` (필터), `Footer`
-   **Apple HIG UX 규칙**:
    -   **Calm Luxury**: 그리드 형태의 프로 카드 간 충분한 여백, 불필요한 장식 제거.
    -   **Data Clarity**: 각 프로 카드에 핵심 정보(이름, 지역, 평점, 주요 태그)를 간결하게 표시하여 한눈에 스캔 가능하게 합니다.
    -   **Single Core Action**: 각 프로 카드의 클릭은 해당 프로의 상세 프로필 페이지로 이동하는 단일 액션에 집중합니다.

```text
+----------------------------------------------------------------------------------------------------------------+
| [Header] LOGO                  [전체 프로] [소개] [문의]                         [Button: 프로 등록]             |
+----------------------------------------------------------------------------------------------------------------+
|                                                                                                                |
|  Main Content Area (max-w-screen-xl, px-space-8)                                                               |
|  +------------------------------------------------------------------------------------------------------------+ |
|  | [Input: 검색]                                                                                              | |
|  +------------------------------------------------------------------------------------------------------------+ |
|  |                                                                                                            | |
|  | [Tag: #비거리] [Tag: #필드레슨] [Tag: #LPGA] ... (필터 태그 목록)                                              |
|  |                                                                                                            | |
|  | +--------------------------------+  +--------------------------------+  +--------------------------------+ |
|  | | [Card: 프로 스니펫]            |  | [Card: 프로 스니펫]            |  | [Card: 프로 스니펫]            | |
|  | |  [프로 사진]                   |  |  [프로 사진]                   |  |  [프로 사진]                   | |
|  | |  [H3] 프로 이름                |  |  [H3] 프로 이름                |  |  [H3] 프로 이름                | |
|  | |  [Body] 주요 경력              |  |  [Body] 주요 경력              |  |  [Body] 주요 경력              | |
|  | |  [Tag: #LPGA] [Tag: #강남]     |  |  [Tag: #PGA] [Tag: #청담]      |  |  [Tag: #KLPGA] [Tag: #판교]    | |
|  | +--------------------------------+  +--------------------------------+  +--------------------------------+ |
|  |                                                                                                            | |
|  | (반복)                                                                                                     | |
|  |                                                                                                            | |
|  | [Button: 더 보기] (Pageination or Load More)                                                               | |
|  +------------------------------------------------------------------------------------------------------------+ |
|                                                                                                                |
+----------------------------------------------------------------------------------------------------------------+
| [Footer] (회사 정보, 링크 등)                                                                                  |
+----------------------------------------------------------------------------------------------------------------+
```

### 1.2. Screen 1.2: 프로 상세 프로필

-   **핵심 목적**: 선택된 프로의 모든 상세 정보를 확인하고, 레슨 문의를 결정합니다.
-   **한 줄 요약**: "매거진 표지처럼 아름답게 전시된 프로의 모든 것"
-   **사용 핵심 컴포넌트**: `Header` (뒤로 가기 버튼, 홈 링크), `Card` (정보 섹션), `Tag` (인증/전문 분야), `Button` (Primary CTA: "문의하기"), `Footer`
-   **Apple HIG UX 규칙**:
    -   **Magazine-First**: 고품질 이미지/영상이 전면에 배치됩니다. 스크롤을 통해 프로의 매력을 탐색하는 경험을 제공합니다.
    -   **Visual-First Storytelling**: 텍스트 설명보다 사진, 영상, 인포그래픽을 통해 프로의 가치를 전달합니다.
    -   **Single Core Action**: 페이지 하단에 고정된 "문의하기" 버튼이 유일한 핵심 CTA로, 다른 방해 없이 문의 액션에 집중하게 합니다.

```text
+----------------------------------------------------------------------------------------------------------------+
| [Header] < 뒤로              [LOGO]                                                [Primary CTA: 프로 등록]       |
+----------------------------------------------------------------------------------------------------------------+
|                                                                                                                |
|  Main Content Area (max-w-screen-xl, px-space-8)                                                               |
|  +------------------------------------------------------------------------------------------------------------+ |
|  | [대형 프로 이미지/영상]                                                                                    | |
|  |                                                                                                            | |
|  | +--------------------------------+                                                                       | |
|  | | [Card: 프로 핵심 정보]         |                                                                       | |
|  | |  [H1] 프로 이름                |                                                                       | |
|  | |  [Body] 소개 요약              |                                                                       | |
|  | |  [Tag: KPGA] [Tag: 비거리 교정]|                                                                       | |
|  | |  [Tag: ★ 4.9]                |                                                                       | |
|  | +--------------------------------+                                                                       | |
|  |                                                                                                            | |
|  | [H2] 프로 상세 소개                                                                                        | |
|  | [Body] 상세 바이오그래피                                                                                   | |
|  |                                                                                                            | |
|  | [H2] 커리큘럼                                                                                              | |
|  | [Card: 타임라인형 커리큘럼]                                                                                | |
|  |                                                                                                            | |
|  | [H2] 레슨 후기                                                                                             | |
|  | [Card: 리뷰 목록]                                                                                          | |
|  +------------------------------------------------------------------------------------------------------------+ |
|                                                                                                                |
| [Sticky Footer with Button]                                                                                    |
| +------------------------------------------------------------------------------------------------------------+ |
| | [Button: 문의하기]                                                                                         | |
| +------------------------------------------------------------------------------------------------------------+ |
|                                                                                                                |
+----------------------------------------------------------------------------------------------------------------+
```

### 1.3. Screen 1.3: 문의 시작 (채팅방 생성)

-   **핵심 목적**: 골퍼가 선택한 프로에게 첫 메시지를 보내 채팅방을 생성하고, Lead로 기록합니다.
-   **한 줄 요약**: "프로와 첫 대화를 시작하는 간결한 화면"
-   **사용 핵심 컴포넌트**: `Header` (뒤로 가기 버튼, 프로 이름), `Input` (텍스트 영역), `Button` (전송), `Chat Bubble` (예상)
-   **Apple HIG UX 규칙**:
    -   **Cognitive Ease**: 복잡한 입력 필드 없이, 첫 메시지 입력에 집중합니다.
    -   **Transparent Interaction**: 채팅방이 생성되는 순간이 명확하게 인지됩니다.
    -   **Focus on Content**: 불필요한 UI 요소 없이, 메시지 입력과 전송에만 집중합니다.

```text
+---------------------------------------+
| [Header] < 프로 이름                  [X] |
+---------------------------------------+
|                                       |
|  Main Content Area (max-w-screen-sm)  |
|  +-----------------------------------+  |
|  | [Chat Bubble: 시스템 메시지]      |  |
|  |  "프로님께 문의를 시작합니다."      |  |
|  |                                   |  |
|  | [Input: 메시지 입력창]            |  |
|  |   "레슨에 대해 궁금한 점을 남겨주세요." |  |
|  +-----------------------------------+  |
|                                       |
| [Button: 전송] (Primary CTA)          |
+---------------------------------------+
```

## 2. Pro Onboarding: Profile Creation → Portfolio Upload → Toggle Public Status (프로: 온보딩)

**플로우 요약**: 프로가 TEE:UP에 가입하여 자신의 프로필을 생성하고, 포트폴리오를 업로드한 후, 프로필을 공개 상태로 전환하는 과정입니다.

### 2.1. Screen 2.1: 프로필 기본 정보 입력

-   **핵심 목적**: 프로의 이름, 경력, 지역 등 필수 기본 정보를 입력합니다.
-   **한 줄 요약**: "단계별로 진행되는 프로필 기본 정보 입력"
-   **사용 핵심 컴포넌트**: `Header` (진행 상태 표시), `Input` (다양한 타입), `Button` (다음 단계, 이전 단계)
-   **Apple HIG UX 규칙**:
    -   **Single Core Action per Screen**: 각 화면에서 하나의 주요 목표(예: 기본 정보 입력)에 집중합니다.
    -   **Clear Progress Indicator**: 헤더에 "1/3단계"와 같은 명확한 진행 상태를 표시하여 사용자가 현재 위치와 남은 단계를 인지하게 합니다.
    -   **Cognitive Ease**: 복잡한 필드를 여러 화면으로 나누어 인지 부하를 줄입니다.

```text
+---------------------------------------+
| [Header] LOGO        [1/3단계]        |
+---------------------------------------+
|                                       |
|  Main Content Area (max-w-screen-md)  |
|  +-----------------------------------+  |
|  | [H2] 프로필 기본 정보              |  |
|  |                                   |  |
|  | [Input: 이름]                     |  |
|  | [Input: 소속]                     |  |
|  | [Input: 주요 레슨 지역]            |  |
|  | [Input: 경력 요약]                |  |
|  |                                   |  |
|  | [Button: 다음 단계] (Primary CTA) |  |
|  | [Button: 이전] (Secondary)        |  |
|  +-----------------------------------+  |
|                                       |
+---------------------------------------+
```

### 2.2. Screen 2.2: 포트폴리오 (사진/영상) 업로드

-   **핵심 목적**: 프로필을 더욱 풍부하게 만들 고품질 사진 및 스윙 영상 파일을 업로드합니다.
-   **한 줄 요약**: "매력적인 프로필을 위한 시각 자료 업로드"
-   **사용 핵심 컴포넌트**: `Header` (진행 상태 표시), `Input` (파일 업로드), `Button` (업로드, 다음 단계, 건너뛰기)
-   **Apple HIG UX 규칙**:
    -   **Visual-First**: 업로드 영역을 크게 제시하고, 미리보기를 통해 업로드될 콘텐츠를 명확히 보여줍니다.
    -   **Intuitive Interaction**: 드래그 앤 드롭과 같은 직관적인 업로드 방식을 제공합니다.
    -   **Flexible Path**: "건너뛰기" 옵션을 통해 사용자가 부담 없이 다음 단계로 넘어갈 수 있도록 합니다.

```text
+---------------------------------------+
| [Header] LOGO        [2/3단계]        |
+---------------------------------------+
|                                       |
|  Main Content Area (max-w-screen-md)  |
|  +-----------------------------------+  |
|  | [H2] 포트폴리오 업로드              |  |
|  | [Body] 프로필 사진과 스윙 영상을 업로드 해주세요. |
|  |                                   |  |
|  | +-------------------------------+ |  |
|  | | [Input: 드래그 앤 드롭 영역]    | |  |
|  | |  + 파일 선택 버튼             | |  |
|  | +-------------------------------+ |  |
|  |                                   |  |
|  | [Image/Video 미리보기]            |  |
|  |                                   |  |
|  | [Button: 다음 단계] (Primary CTA) |  |
|  | [Button: 건너뛰기] (Ghost)        |  |
|  +-----------------------------------+  |
|                                       |
+---------------------------------------+
```

### 2.3. Screen 2.3: 프로필 공개 상태 전환

-   **핵심 목적**: 생성된 프로필을 최종 검토하고, TEE:UP 플랫폼에 공개할지 여부를 설정합니다.
-   **한 줄 요약**: "프로필 검토 및 공개 설정"
-   **사용 핵심 컴포넌트**: `Header` (완료 메시지), `Card` (프로필 요약), `Toggle Switch` (공개/비공개), `Button` (프로필 미리보기, 대시보드로 이동)
-   **Apple HIG UX 규칙**:
    -   **Calm Control**: 사용자가 자신의 프로필 공개 여부를 명확하고 쉽게 제어할 수 있도록 합니다.
    -   **Transparent Options**: 토글 스위치를 통해 공개 상태의 변경이 즉각적으로 시각화됩니다.
    -   **Confirmation**: 주요 액션(공개 전환) 전 요약 정보를 제공하여 최종 결정을 돕습니다.

```text
+---------------------------------------+
| [Header] LOGO        [완료]           |
+---------------------------------------+
|                                       |
|  Main Content Area (max-w-screen-md)  |
|  +-----------------------------------+  |
|  | [H2] 프로필이 성공적으로 생성되었습니다! |
|  |                                   |  |
|  | [Card: 프로필 요약]               |  |
|  |  [프로 사진] [프로 이름] [핵심 정보]|  |
|  |                                   |  |
|  | [Toggle Switch: 프로필 공개]      |  |
|  |  "지금 프로필을 공개하시겠습니까?"   |  |
|  |                                   |  |
|  | [Button: 대시보드로 이동] (Primary CTA)|  |
|  | [Button: 프로필 미리보기] (Secondary) |  |
|  +-----------------------------------+  |
|                                       |
+---------------------------------------+
```

## 3. Pro Dashboard: Today's Lead List → Detail View → Follow-up Action (프로: 리드 관리)

**플로우 요약**: 프로가 자신의 대시보드에서 새로운 문의(Lead)를 확인하고, 상세 정보를 조회한 후, 골퍼와 채팅을 시작하여 후속 조치를 취하는 과정입니다.

### 3.1. Screen 3.1: 프로 대시보드 (리드 목록)

-   **핵심 목적**: 프로가 자신에게 도착한 새로운 레슨 문의(Lead) 목록을 한눈에 확인하고 관리합니다.
-   **한 줄 요약**: "새로운 기회를 포착하는 리드 관리 대시보드"
-   **사용 핵심 컴포넌트**: `Header` (알림, 프로필), `Card` (통계 요약), `Table` (리드 목록), `Tag` (리드 상태), `Button` (CTA per lead)
-   **Apple HIG UX 규칙**:
    -   **Data Clarity**: 주요 통계(총 리드, 매칭률 등)와 리드 목록이 명확하게 시각화됩니다.
    -   **Calm Control**: 필터링 및 정렬 옵션을 통해 리드 목록을 쉽게 관리할 수 있습니다.
    -   **Scannable Information**: 리드 목록은 간결한 테이블 형태로 제공되어 핵심 정보를 빠르게 파악할 수 있습니다.

```text
+----------------------------------------------------------------------------------------------------------------+
| [Header] LOGO                  [알림] [프로필] [설정]                                                          |
+----------------------------------------------------------------------------------------------------------------+
|                                                                                                                |
|  Main Content Area (max-w-screen-xl, px-space-8)                                                               |
|  +------------------------------------------------------------------------------------------------------------+ |
|  | [H1] 프로 대시보드                                                                                         | |
|  |                                                                                                            | |
|  | +-------------------------+ +-------------------------+ +-------------------------+                      | |
|  | | [Card: 총 리드]         | | [Card: 이번 달 문의]    | | [Card: 매칭률]          |                      | |
|  | |  [Metric: 50+]          | |  [Metric: 5]             | |  [Metric: 40%]          |                      | |
|  | +-------------------------+ +-------------------------+ +-------------------------+                      | |
|  |                                                                                                            | |
|  | [H2] 오늘의 리드                                                                                           | |
|  |                                                                                                            | |
|  | +--------------------------------------------------------------------------------------------------------+ |
|  | | [Table: 리드 목록]                                                                                     | |
|  | |  Header: 골퍼 이름, 문의 날짜, 상태, 액션                                                               | |
|  | |  Row 1: [골퍼 A], [2025-12-09], [Tag: 신규], [Button: 확인하기]                                          | |
|  | |  Row 2: [골퍼 B], [2025-12-08], [Tag: 진행중], [Button: 확인하기]                                        | |
|  | +--------------------------------------------------------------------------------------------------------+ |
|  +------------------------------------------------------------------------------------------------------------+ |
|                                                                                                                |
+----------------------------------------------------------------------------------------------------------------+
| [Footer] (회사 정보, 링크 등)                                                                                  |
+----------------------------------------------------------------------------------------------------------------+
```

### 3.2. Screen 3.2: 리드 상세 조회

-   **핵심 목적**: 특정 문의(Lead)를 선택하여 골퍼의 상세 정보와 문의 내용을 확인합니다.
-   **한 줄 요약**: "골퍼 문의의 모든 정보를 한눈에"
-   **사용 핵심 컴포넌트**: `Header` (뒤로 가기 버튼, 리드 ID), `Card` (골퍼 정보, 문의 내용), `Button` (Primary CTA: "채팅 시작하기")
-   **Apple HIG UX 규칙**:
    -   **Information Hierarchy**: 골퍼의 핵심 정보(프로필, 레벨)와 문의 내용을 명확한 계층 구조로 배치합니다.
    -   **Single Core Action**: 페이지 하단에 "채팅 시작하기" 버튼을 배치하여 다음 단계로의 전환을 유도합니다.
    -   **Cognitive Ease**: 불필요한 정보 없이 문의 해결에 필요한 정보만 제시합니다.

```text
+---------------------------------------+
| [Header] < 리드 상세 (ID: #12345)    [X] |
+---------------------------------------+
|                                       |
|  Main Content Area (max-w-screen-md)  |
|  +-----------------------------------+  |
|  | [Card: 골퍼 정보]                 |  |
|  |  [골퍼 프로필 이미지]               |  |
|  |  [H3] 골퍼 이름 (레벨)             |  |
|  |  [Body] 평균 핸디캡, 레슨 경험 등   |  |
|  |                                   |  |
|  | [Card: 문의 내용]                 |  |
|  |  [Body] "안녕하세요, 프로님! 비거리..." |  |
|  |                                   |  |
|  | [Button: 채팅 시작하기] (Primary CTA)|  |
|  +-----------------------------------+  |
|                                       |
+---------------------------------------+
```

### 3.3. Screen 3.3: Follow-up 액션 (채팅방 진입)
